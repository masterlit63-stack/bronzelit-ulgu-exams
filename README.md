# Бронзелит — экспресс-подготовка к вступительным экзаменам УлГУ

Мини-лендинг проекта **«Бронзелит»**: продажа готовых учебных материалов
для быстрой подготовки к вступительным экзаменам УлГУ по **русскому языку,
химии и биологии**.

## О проекте

Одностраничный сайт (`index.html`) без внешних зависимостей: весь CSS
встроен в файл, скриптов и сторонних библиотек нет. Страница адаптивна и
корректно отображается на телефоне.

Материалы рассчитаны на абитуриентов, которым нужно за 1–3 дня восстановить
школьную программу и потренироваться на заданиях, похожих на вступительный
экзамен. Для каждого предмета есть конспект, упражнения, тесты, ответы и
пробные варианты.

Материалы являются независимыми учебными пособиями и не являются официальными
материалами УлГУ. Они не гарантируют конкретный балл: результат зависит от
исходных знаний и работы ученика.
