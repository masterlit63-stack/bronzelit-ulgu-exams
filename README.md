# Бронзелит — экспресс-повторение перед вступительными экзаменами

Мини-лендинг проекта **«Бронзелит»**: независимые учебные материалы
для экспресс-повторения перед вступительными экзаменами по **русскому языку,
химии и биологии**. Материалы ориентированы на темы из открыто опубликованных
программ вступительных испытаний по соответствующим предметам.

## О проекте

Одностраничный сайт (`index.html`) без внешних зависимостей: весь CSS
встроен в файл, скриптов и сторонних библиотек нет. Страница адаптивна и
корректно отображается на телефоне.

Материалы рассчитаны на абитуриентов, которым нужно за 1–3 дня восстановить
школьную программу и потренироваться на заданиях, похожих на вступительный
экзамен. Для каждого предмета есть конспект, упражнения, тесты, ответы и
пробные варианты.
